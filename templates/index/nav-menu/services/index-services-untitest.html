What is Unit Test?

This is a type of software testing where individual units or components of a software are tested.

What is UnitBench.Py?

UnitBench.Py is an intelligent framework that will automate test cases writing for that source code which will be written in BenchPy IDE.

Why UnitBench.Py?

It is reduces developer's overhead and saves time by writing and managing automatic test case for the source code. Which improves development throughput.
